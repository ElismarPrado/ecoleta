# ecoleta
Projeto de Aplicativo para encontrar pontos de coleta de reciclaveis. Foram desenvolvidos um aplicativo para encontrar os pontos de coleta e um sistema web para estabelecimentos cadastrarem-se como pontos coletadores, ambos utilizam uma api rest como back-end. foi utilizado as tecnologias e linguagens: TypeScript, React.js, React Native e Node.js
